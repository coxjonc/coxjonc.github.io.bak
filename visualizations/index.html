Visualizations

I’m addicted to building graphs with D3 and making maps with Leaflet. You can see the terrible consequences of my addiction here.

Austin restaurant health scores

A LeafletJS map with marker clusters and search functionality that plots the locations and scores of all the establishments (restaurants, hotels, and schools) in the city’s health inspection database. Data obtained from data.austintexas.gov, cleaned with OpenRefine, converted to JSON with a Python script.

Dynamic weather graph

A React app that populates a select box with the list of cities from Weather Underground’s autocomplete API. When a city is selected it makes another AJAX call to Weather Underground, then generates a D3 line graph of the hourly forecast for that city for the next three days. The graph is interactive - hover over it to see a tooltip appear on the point closest to your mouse. Source on GitHub.

Minimum wage by state

A sortable D3 bar chart of minimum wage in the U.S., by state.

NASA yearly budget since 1960

An interactive line graph built with D3 that creates a tooltip with information about the datapoint closest to the user’s mouse.
